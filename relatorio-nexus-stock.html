🎯 Visão Geral do Sistema

O Nexus Stock é um sistema completo de gestão de armazém desenvolvido com tecnologias modernas, projetado para controlar entrada, saída e localização de produtos em tempo real. O sistema utiliza leitura de código de barras, NFC e cálculo hierárquico de unidades para garantir precisão no estoque.

🚀 Funcionalidades Destacadas

🎯 Cálculo Hierárquico de Estoque

Sistema inteligente que calcula automaticamente as quantidades em diferentes níveis. Quando você estoca 1 CAIXA MASTER com 36 unidades, o sistema automaticamente calcula:

  • 1 CAIXA MASTER = 36 DISPLAYS = 36 UNIDADES
  • 1 BOX/DISPLAY = X UNIDADES (configurável)
  • Agrupamento por SKU mostra tudo consolidado
  • Visualização separada por tipo de unidade

📱 Scan de Código de Barras

Fluxo otimizado para entrada de produtos:

  • Escaneie ou digite o código de barras
  • Busca automática ao pressionar Enter
  • Se não encontrar, modal abre para criar produto
  • Localização e bin sugeridos automaticamente
  • Confirmar entrada sem sair da tela

🌓 Interface Moderna com Temas

Design premium com suporte a temas:

  • Tema escuro com gradientes slate-950 → slate-900
  • Tema claro com gradientes gray-50 → white
  • Toggle com um clique
  • Transições suaves de 300ms
  • Efeitos glassmorphism
  • Botões com gradientes coloridos

📍 Localizações Human-Readable

Fim dos UUIDs confusos! Agora as localizações são apresentadas de forma clara:

  • Z Zona (azul)
  • C Corredor (verde)
  • S Seção (roxo)
  • P Prateleira (laranja)
  • Badges coloridos para fácil identificação
  • IDs truncados para melhor leitura

🔍 Consulta Inteligente de Estoque

Encontre qualquer produto facilmente:

  • Agrupamento automático por SKU

  • Visualização de quantidades em 3 níveis
  • Localizações de cada produto
  • Contadores consolidados por SKU
  • Badges mostrando tipo de unidade
