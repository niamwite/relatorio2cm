## 📋 Passo a Passo para Ativar GitHub Pages

### 1️⃣ Acesse o Repositório

👉 **Clique aqui:** https://github.com/niamwite/relatorio2cm

### 2️⃣ Vá em Settings

- No topo do repositório, clique na aba **Settings** (⚙️)
- É a última aba no menu superior

### 3️⃣ Configure Pages

- No menu lateral esquerdo, role até encontrar **Pages**
- Clique em **Pages**

### 4️⃣ Ative o Site

Na seção **Build and deployment**:

**Source:**
- Selecione: **Deploy from a branch**

**Branch:**
- Branch: **main**
- Folder: **/** (root directory)
- Clique em **Save**

### 5️⃣ Aguarde o Deploy

- Aguarde 1-3 minutos
- Atualize a página
- Você verá seu link no topo:
  ```
  https://niamwite.github.io/relatorio2cm/
  ```

## 🐛 Troubleshooting

### Site não aparece?
- Aguarde até 5 minutos (GitHub precisa processar)
- Limpe o cache do navegador (Ctrl + F5)

### Erro 404?
- Verifique se selecionou a branch **main** (não master)
- Verifique se a pasta é **/** (root)

### Links quebrados?
- Deixe um comentário aqui se algum link não funcionar
